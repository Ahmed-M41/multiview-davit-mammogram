# Data

Raw images are **not included** in this repository. Follow the instructions below to obtain each dataset.

## CBIS-DDSM

Download from The Cancer Imaging Archive (TCIA):

- URL: https://www.cancerimagingarchive.net/collection/cbis-ddsm/
- DOI: 10.7937/K9/TCIA.2016.7O02S9CY
- Licence: CC BY 3.0

After downloading, place the DICOM files under `data/raw/cbis_ddsm/` and the description CSVs (mass + calc) in the same directory.

## INbreast

Download from the original source (for academic research):

- DOI: 10.1016/j.acra.2011.09.014
- Contact: Breast Research Group, INESC Porto / Centro Hospitalar de S. João

Place the DICOM files under `data/raw/inbreast/`.

## Preprocessed Data (FigShare)

Preprocessed image archives (512×512 JPEG), dataset metadata CSVs, the BI-RADS codebook, and per-image model predictions are archived at:

- DOI: [10.6084/m9.figshare.32149885](https://doi.org/10.6084/m9.figshare.32149885)

Download and extract `cbis_ddsm_preprocessed_224px.zip` and `inbreast_preprocessed_512px.zip` into `data/processed/`.

## Expected Directory Structure

```
data/
├── raw/
│   ├── cbis_ddsm/          # Raw DICOMs + description CSVs
│   └── inbreast/           # Raw DICOMs
├── processed/
│   ├── cbis_ddsm/          # Preprocessed 512×512 JPEGs
│   └── inbreast/           # Preprocessed 512×512 JPEGs
└── splits/                 # Auto-generated by CLI (gitignored)
    ├── cbis_train.csv
    ├── cbis_val.csv
    └── cbis_test.csv
```
